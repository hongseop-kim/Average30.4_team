# Average30.4_team

### 프로젝트 설명
게임 산업의 발전으로 새로운 게임중개 플랫폼 개발.
특별한 기간에 CI/CD를 활용하여 정해진 기간 자동 Rolling Update 및 Roll Back.
Amazon EKS(Elastic Kubernetes Service)를 통해 쿠버네티스 클러스터를 구축 후 오토스케일링, 로드밸런싱을 통한 부하 분산이 되도록 구현.
